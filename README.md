A high performance serial receiver implementation for the duopinball i-pad controller.
- This software fixes the persistent lag that is caused by the SerialPort.DataReceived event handler, and configure the seriaL port for best performance.
- translates input from the duopinball controller into keyboard IO (lshift and rshift keys), or XBox 360 controller IO (Ltrigger Rtrigger).
- Inspired by [duopinball on windows](https://digitalpinballfans.com/threads/duo-pinball-controller-can-be-made-to-work-on-a-windows-tablet.6953/)
