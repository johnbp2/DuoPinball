A high performance serial receiver implementation for the duopinball i-pad controller.
This lets you translate input into keyboard IO (lshift and rshift keys), or XBox 360 controller IO (Ltrigger Rtrigger).
Inspired by [duopinball on windows](https://digitalpinballfans.com/threads/duo-pinball-controller-can-be-made-to-work-on-a-windows-tablet.6953/)
